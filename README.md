## Installation

IcedCoffee is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'IcedCoffee', :git => 'https://github.com/riley-usagi/IcedCoffee.git', :tag => '1.1.2'
```
